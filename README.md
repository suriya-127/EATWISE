# EatWise - Food Ordering Application

## Project Overview

**EatWise** is a modern web-based food ordering platform designed to bridge the gap between customers and restaurants. The application provides a seamless interface for customers to browse menus, place orders, and track their food, while enabling restaurants to manage their operations efficiently. Built with responsive web technologies, EatWise delivers an intuitive user experience across all devices.

## Key Features

### For Customers
- **Easy Login & Registration**: Secure customer authentication with a smooth onboarding process
- **Interactive Dashboard**: Browse available restaurants and menu items with real-time updates
- **Order Management**: Place orders, track delivery status, and manage order history
- **User-Friendly Interface**: Responsive design optimized for desktop, tablet, and mobile devices

### For Restaurants
- **Restaurant Admin Portal**: Dedicated login for restaurant managers and staff
- **Order Management System**: View incoming orders and manage order status
- **Menu Management**: Update menu items and availability in real-time
- **Restaurant Dashboard**: Comprehensive overview of business metrics and operations
